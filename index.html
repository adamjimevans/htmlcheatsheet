Welcome to my HTML cheat sheet, designed to provide all you need to create some cool conent using html.

General HTML

HTML tags

Tag Name Description
<html> html Sets the document as a html page. All content wrapped by this
<head> Page Header Add the title of your page and link stylesheets and fonts here
<body> Body The content of your page is wrapped here
<h(1,2,3,4,5 or 6)> Text Header Insert a header. The number indicates the size, 1 being the largest.
<p> Paragraph Insert text content onto the page
<div> Division Divide sections of your page. You can style each division uniquely.
<img> Image Present an image on the page

HTML attributes

Attribute Name Description
href Reference Use this to link conent eg images from a source
class Class Give a tag a class so you can refer to it for unique styling

HTML tables

Table tags

Tag Name Description
<table> Table Insert a table. All table code wrapped in this tag.
<thead> Header Add headers to each collumn
<tbody> Body Add the main data to the table
<tr> Row Add a new row to the table
<td> Data Add data to the table
<tfoot> Footer Add a footer to the bottom of the table

Table attributes

Attribute Name Description
colspan Collumn span Define how many collumns a data entry should occupy
rowspan Row span Define how many rows a data entry should occupy
